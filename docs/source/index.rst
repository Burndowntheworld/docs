Welcome to BurnLang documentation!
===================================

**Burn** is a coding language made by burndowntheworld.
It's a c#, python, and some of my own idea hybrid.
It enables the user to make simple quick and effective code.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
